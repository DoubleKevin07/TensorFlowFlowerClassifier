# Usage
1. Install Anaconda
2. If you don't have a TensorFlow environment, run the following commands, through Anaconda.
```
conda create -n tf tensorflow
conda activate tf
conda install matplotlib
classify.py
```
3. Enter `classify.py` into the command line.
